# 投票系统 - 宝塔面板部署教程

## 一、准备工作

### 1.1 服务器要求
- 操作系统：Ubuntu 20.04/22.04 或 CentOS 7/8
- 内存：至少 1GB（推荐 2GB 以上）
- 硬盘：至少 10GB

### 1.2 需要准备的信息
- 域名（已解析到服务器IP）
- Supabase 数据库信息（在 https://supabase.com 创建项目获取）
  - DATABASE_URL（数据库连接字符串）
  - SUPABASE_URL
  - SUPABASE_ANON_KEY

---

## 二、安装宝塔面板

### 2.1 Ubuntu 安装命令
```bash
wget -O install.sh https://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh ed8484bec
```

### 2.2 CentOS 安装命令
```bash
yum install -y wget && wget -O install.sh https://download.bt.cn/install/install_6.0.sh && sh install.sh ed8484bec
```

### 2.3 安装完成后
安装完成后会显示：
- 宝塔面板地址（如：http://你的IP:8888/xxxxxxxx）
- 用户名
- 密码

**请保存这些信息！**

---

## 三、宝塔面板配置

### 3.1 登录宝塔面板
1. 浏览器访问宝塔面板地址
2. 输入用户名和密码登录

### 3.2 安装软件（在"软件商店"中安装）
必须安装以下软件：
- **Nginx** 1.24+
- **PM2管理器** 4.5+（用于运行Node.js应用）
- **Node.js版本管理器**（安装Node.js 18+）

### 3.3 安装Node.js
1. 点击"软件商店"
2. 找到"Node版本管理器"
3. 点击"设置"
4. 安装 Node.js 18.x 或 20.x

---

## 四、上传项目文件

### 4.1 创建网站目录
```bash
mkdir -p /www/wwwroot/vote-system
```

### 4.2 上传源码
**方式一：通过宝塔文件管理器上传**
1. 点击左侧"文件"
2. 进入 `/www/wwwroot/vote-system` 目录
3. 点击"上传"，上传源码压缩包
4. 右键压缩包，选择"解压"

**方式二：通过Git拉取（推荐）**
如果源码在GitHub/Gitee：
```bash
cd /www/wwwroot
git clone 你的仓库地址 vote-system
```

## 五、配置环境变量

### 5.1 创建 .env 文件
在项目根目录创建 `.env` 文件：

```bash
cd /www/wwwroot/vote-system
nano .env
```

### 5.2 填入以下内容（根据你的实际情况修改）
```env
# 数据库配置（从Supabase获取）
DATABASE_URL="postgresql://postgres.xxxxx:密码@aws-0-ap-southeast-1.pooler.supabase.com:6543/postgres"

SUPABASE_URL="https://xxxxx.supabase.co"
SUPABASE_ANON_KEY="你的anon_key"

# 服务配置
PORT=5000
NODE_ENV=production

# 域名（改成你的域名）
COZE_PROJECT_DOMAIN_DEFAULT="你的域名"
```

按 `Ctrl+O` 保存，按 `Ctrl+X` 退出。

## 七、PM2启动服务

### 7.1 通过宝塔PM2管理器启动
1. 点击"软件商店" → "PM2管理器" → "设置"
2. 点击"添加项目"
3. 填写：
   - 项目名称：vote-system
   - 启动文件：/www/wwwroot/vote-system/node_modules/next/dist/bin/next
   - 运行目录：/www/wwwroot/vote-system
   - 启动参数：start
4. 点击"提交"

### 7.2 或通过命令行启动
```bash
cd /www/wwwroot/vote-system
pm2 start "pnpm start" --name vote-system
pm2 save
pm2 startup
```

## 八、Nginx配置

### 8.1 在宝塔创建网站
1. 点击"网站" → "添加站点"
2. 填写：
   - 域名：你的域名（如 vote.example.com）
   - PHP版本：纯静态
   - 点击"提交"

### 8.2 配置反向代理
1. 点击刚创建的网站 → "反向代理" → "添加反向代理"
2. 填写：
   - 代理名称：vote-system
   - 目标URL：http://127.0.0.1:5000
   - 发送域名：$host
3. 点击"提交"

### 8.3 或手动修改Nginx配置
点击网站 → "配置文件"，在 `server` 块内添加：

```nginx
location / {
    proxy_pass http://127.0.0.1:5000;
    proxy_http_version 1.1;
    proxy_set_header Upgrade $http_upgrade;
    proxy_set_header Connection 'upgrade';
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    proxy_set_header X-Forwarded-Proto $scheme;
    proxy_cache_bypass $http_upgrade;
}
```

保存后点击"重载配置"。

## 十二、常见问题

### Q1: 访问显示502错误
- 检查PM2服务是否正常运行
- 检查5000端口是否被占用

```bash
pm2 list
netstat -tlnp | grep 5000
```

### Q2: 数据库连接失败
- 检查 .env 文件中的数据库配置是否正确
- 检查Supabase项目是否正常运行

### Q3: 样式不显示
- 重新构建项目：`pnpm build`
- 清除浏览器缓存

### Q4: PM2服务自动停止
- 查看日志：`pm2 logs vote-system`
- 检查内存是否不足
